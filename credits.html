6. 開発支援ツール

本アプリケーションの開発において、以下のツール・サービスを利用しました：

  • Node.js (v18.0.0以上) - JavaScriptランタイム環境
  • npm - パッケージマネージャー
  • nodemon (v3.0.1) - 開発時の自動再起動ツール
  • Docker - コンテナ化技術（開発・デプロイ環境）
  • GitHub - ソースコード管理・バージョン管理
  • Claude (Anthropic) - AI開発支援
  • Cursor IDE - 統合開発環境

7. デプロイメント環境

本アプリケーションは、以下のプラットフォームでのデプロイに対応しています：

  • Railway - クラウドプラットフォーム
  • Render - クラウドプラットフォーム
  • Heroku - クラウドプラットフォーム
  • Docker Compose - コンテナオーケストレーション
  • 自前サーバー（VPS等） - Node.js直接実行

8. データベース

本アプリケーションは、以下のデータベースをサポートしています：

  • SQLite3 (v5.1.6) - 軽量データベース（開発・小規模運用）
  • PostgreSQL - リレーショナルデータベース（本番環境推奨）
  • Supabase - PostgreSQLベースのBaaS（バックエンドとしてのサービス）

9. セキュリティ機能

本アプリケーションは、以下のセキュリティ機能を実装しています：

  • JWT認証 - トークンベースの認証システム
  • パスワードハッシュ化 - bcryptによる安全なパスワード保存
  • レート制限 - ブルートフォース攻撃対策
  • HTTPS強制 - 暗号化通信の強制（本番環境）
  • セキュリティヘッダー - HelmetによるHTTPヘッダー保護
  • 入力検証 - SQLインジェクション対策
  • CORS設定 - クロスオリジンリクエストの制御
