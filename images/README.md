# 画像フォルダ

このフォルダには、アプリケーションで使用する画像ファイルを配置します。

## ログイン背景画像の配置方法

1. ダウンロードフォルダから `Whisk_c2b51992ca2b95498a24c927afe6d5b2eg` ファイルを探す

2. このファイルを `images` フォルダにコピー

3. ファイル名を `login-bg.png` にリネーム
   - 画像がJPEGの場合は `login-bg.jpg` にリネーム
   - その場合、`index.html` の 545行目も修正してください：
     ```css
     url('images/login-bg.jpg')  /* .png を .jpg に変更 */
     ```

4. ファイルが正しく配置されたか確認：
   ```bash
   ls -la images/login-bg.*
   ```

5. ブラウザでログイン画面をリロードして確認

## 画像のサイズについて

- 背景画像は自動的に画面全体にフィットします（`cover`）
- 推奨サイズ: 1920x1080 以上
- ファイルサイズ: 500KB以下を推奨（読み込み速度のため）

## 画像が表示されない場合

1. ファイル名が正しいか確認（大文字小文字も区別されます）
2. ファイルパスが正しいか確認
3. ブラウザのキャッシュをクリア（Ctrl+Shift+R / Cmd+Shift+R）
4. ブラウザのコンソールでエラーを確認
